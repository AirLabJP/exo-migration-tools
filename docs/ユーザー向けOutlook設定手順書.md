## 1. 事前準備

### 1.1 必要なもの

| 項目 | 内容 |
|------|------|
| **Windowsアカウント** | 会社のPCにログイン済みであること |
| **ネットワーク接続** | インターネットに接続されていること |
| **メールアドレス** | システム管理者から通知された新しいメールアドレス |

### 1.2 Outlookがインストールされていることを確認

1. **Windowsのスタートメニュー**を開く
2. 「**Outlook**」と入力して検索
3. **Microsoft Outlook** が表示されることを確認

> **Outlookが見つからない場合**: システム管理者にお問い合わせください。

## 2. Outlookの初回起動と設定

### 2.1 Outlookを起動

1. **スタートメニュー** → **Outlook** をクリック
2. 初回起動時は自動的に設定ウィザードが開始されます

### 2.2 アカウント設定（自動設定）

多くの場合、Outlookは自動的にアカウントを検出します。

1. **「メールアドレスを入力してください」** 画面で、
   自分のメールアドレス（例：`user@example.co.jp`）を入力
2. **「接続」** をクリック

![メールアドレス入力画面](images/outlook_setup_01.png)
※ 画像は参考イメージです

3. **サインイン画面** が表示される場合:
   - 会社のメールアドレスを入力
   - **次へ** をクリック
   - パスワードを入力（会社のWindowsログインパスワードと同じ場合が多い）
   - **サインイン** をクリック

4. **「アカウントが正常に追加されました」** と表示されたら完了

### 2.3 設定完了の確認

1. **「完了」** をクリックしてOutlookを起動
2. 受信トレイが表示されることを確認
3. 画面左側に自分のメールアドレスが表示されていることを確認

## 3. 基本的な使い方

### 3.1 メールの受信

- Outlookは自動的に新しいメールを受信します
- 手動で受信する場合: **送受信** タブ → **すべてのフォルダーを送受信**
- ショートカット: **F9**キー

### 3.2 メールの送信

1. **ホーム** タブ → **新しいメール** をクリック
2. **宛先**: 送信先のメールアドレスを入力
   - 社内の人は名前の一部を入力すると候補が表示されます（GAL検索）
3. **件名**: メールの件名を入力
4. **本文**: メール本文を入力
5. **送信** ボタンをクリック

### 3.3 アドレス帳（GAL）の使い方

社内の連絡先は**グローバルアドレス一覧（GAL）**から検索できます。

1. 新規メール作成画面で **宛先** をクリック
2. **アドレス帳** ウィンドウが開く
3. 上部の検索ボックスに名前の一部を入力
4. 目的の人を選択して **宛先** → **OK**

> **💡 ヒント**: 宛先欄に名前を直接入力すると、自動的にGALから候補が表示されます。

### 3.4 添付ファイルの送信

1. メール作成画面で **メッセージ** タブ → **ファイルの添付**
2. ファイルを選択して **挿入**
3. **送信** をクリック

> **注意**: 外部宛のメールに添付ファイルを付けると、自動的にURLに変換される場合があります（セキュリティ設定による）。

## 4. 便利な設定

### 4.1 署名の設定

1. **ファイル** → **オプション** → **メール** → **署名**
2. **新規作成** をクリックして署名名を入力（例：「標準署名」）
3. 署名の内容を入力:
   ```
   ━━━━━━━━━━━━━━━━━━━━━━
   ○○株式会社 △△部
   山田 太郎（Yamada Taro）
   TEL: 03-XXXX-XXXX
   Email: yamada@example.co.jp
   ━━━━━━━━━━━━━━━━━━━━━━
   ```
4. **新しいメッセージ** のドロップダウンで署名を選択
5. **OK** をクリック

### 4.2 表示名の確認

1. **ファイル** → **アカウント設定** → **アカウント設定**
2. 自分のアカウントを選択 → **変更**
3. **自分の名前** が正しく設定されているか確認
4. 必要に応じて修正 → **次へ** → **完了**

### 4.3 不在時の自動応答（休暇設定）

1. **ファイル** → **自動応答**
2. **自動応答を送信する** を選択
3. 期間を設定（任意）
4. 自動応答メッセージを入力
5. **OK** をクリック

> **注意**: 外部への自動応答は制限されている場合があります。

## 5. トラブルシューティング

### 5.1 Outlookが起動しない

**対処法1: Outlookを再起動**
1. タスクマネージャーを開く（Ctrl+Shift+Esc）
2. 「Outlook」を選択して「タスクの終了」
3. 再度Outlookを起動

**対処法2: PCを再起動**
1. Windowsを再起動して再度試す

### 5.2 メールが送受信できない

**確認事項**:
- インターネット接続があるか
- 画面右下に「接続済み」と表示されているか

**対処法**:
1. **送受信** タブ → **すべてのフォルダーを送受信** を試す
2. しばらく待っても改善しない場合はシステム管理者に連絡

### 5.3 パスワードを求められる

**対処法**:
1. 会社のWindowsログインパスワードを入力
2. **資格情報を記憶する** にチェックを入れる
3. **OK** をクリック

### 5.4 「サーバーに接続できません」と表示される

**対処法**:
1. インターネット接続を確認
2. VPN接続が必要な場合は接続
3. Outlookを再起動
4. 改善しない場合はシステム管理者に連絡

## 6. よくある質問

### Q1. Thunderbirdの過去メールはどうなりますか？

**A**: 過去のメールはThunderbirdのローカルフォルダに残っています。
必要に応じて参照することができます。
過去メールをOutlookに移行したい場合は、「[過去メール移行手順書](ユーザー向け過去メール移行手順書.md)」を参照してください。

### Q2. アドレス帳はどうなりますか？

**A**: 社内の連絡先は **グローバルアドレス一覧（GAL）** から検索できます。
Thunderbirdのローカルアドレス帳は自動では移行されません。
よく連絡する外部の方は、Outlookの連絡先に手動で登録してください。

### Q3. 転送設定はできますか？

**A**: セキュリティ上の理由から、**外部への自動転送は禁止**されています。
社内への転送については、システム管理者にご相談ください。

### Q4. スマートフォンでもメールを見られますか？

**A**: 現在は対象外です。Phase 2以降で対応予定です。

### Q5. Webメール（ブラウザ）でアクセスできますか？

**A**: はい、以下のURLからアクセスできます:
- https://outlook.office.com/

会社のメールアドレスとパスワードでサインインしてください。

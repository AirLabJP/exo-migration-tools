# 過去メール移行手順書（Thunderbird → Outlook）

## はじめに

この手順書は、Thunderbirdに保存されている過去のメールを、Outlookに移行する方法を説明します。

> **注意**: この作業は**任意**です。過去メールをOutlookに移行しなくても、新しいメールの送受信には影響ありません。必要な方のみ実施してください。

## 1. 移行の概要

### 1.1 移行の流れ

```
Thunderbird → EMLファイルにエクスポート → Outlookにインポート
```

### 1.2 所要時間の目安

| メール数 | 目安時間 |
|----------|----------|
| 100通未満 | 約10分 |
| 100〜1,000通 | 約30分 |
| 1,000通以上 | 1時間以上 |

### 1.3 注意事項

- 移行作業中はThunderbirdとOutlookを同時に起動します
- 十分なディスク容量があることを確認してください
- 作業中はPCの電源を切らないでください

## 2. 準備作業

### 2.1 必要なものの確認

- [x] Thunderbirdがインストールされている
- [x] Outlookが設定済みである
- [x] 十分なディスク容量がある（メールデータの2倍程度）

### 2.2 作業用フォルダの作成

1. デスクトップを右クリック → **新規作成** → **フォルダー**
2. フォルダ名を「**メール移行作業**」に変更

---

## 3. 方法A: ImportExportToolsを使った移行

**推奨**: 大量のメールを移行する場合

### 3.1 アドオンのインストール

1. **Thunderbird** を起動
2. メニュー **≡** → **アドオンとテーマ**
3. 検索ボックスに「**ImportExportTools**」と入力
4. 「**ImportExportTools NG**」を見つけて **Thunderbirdへ追加**
5. **今すぐ再起動** をクリック

### 3.2 メールのエクスポート

1. Thunderbirdで移行したいフォルダを右クリック
   （例：受信トレイ、送信済みトレイ）
2. **ImportExportTools NG** → **フォルダ内のすべてのメッセージをエクスポート** → **EML形式**
3. 保存先として「**メール移行作業**」フォルダを選択
4. エクスポートが完了するまで待機

> **💡 ヒント**: フォルダごとにサブフォルダを作成すると整理しやすくなります。
> 例：`メール移行作業\受信トレイ`、`メール移行作業\送信済み`

### 3.3 Outlookへのインポート

1. **Outlook** を起動
2. エクスプローラーで「**メール移行作業**」フォルダを開く
3. Outlookで移行先のフォルダを選択（例：受信トレイ）
4. エクスプローラーからOutlookへEMLファイルを**ドラッグ＆ドロップ**

> **注意**: 一度に大量のファイルをドロップすると時間がかかります。
> 100〜200ファイルずつ分けて実行することをお勧めします。

---

## 4. 方法B: ドラッグ＆ドロップによる移行

**推奨**: 少量のメール（100通未満）を移行する場合

### 4.1 手順

1. **Thunderbird** と **Outlook** を両方起動
2. 両方のウィンドウを並べて表示
3. Thunderbirdで移行したいメールを選択
   - 複数選択: **Ctrl** を押しながらクリック
   - 連続選択: **Shift** を押しながらクリック
   - 全選択: **Ctrl + A**
4. 選択したメールをOutlookの目的のフォルダにドラッグ＆ドロップ

### 4.2 フォルダ単位での移行

1. Thunderbirdでフォルダを右クリック
2. **ImportExportTools NG** → **フォルダ内のすべてのメッセージをエクスポート** → **EML形式**
3. 一時フォルダに保存
4. Outlookへドラッグ＆ドロップ

---

## 5. 移行後の確認

### 5.1 確認事項

| 項目 | 確認方法 |
|------|----------|
| メール数 | Thunderbirdのメール数とOutlookのメール数を比較 |
| 添付ファイル | 重要なメールの添付ファイルが開けるか確認 |
| 日付 | メールの送受信日時が正しく表示されているか |
| 文字化け | 日本語が正しく表示されているか |

### 5.2 移行完了後の整理

1. 作業用フォルダ「**メール移行作業**」を削除
2. Thunderbirdの過去メールは参照用として残すことをお勧めします

## 6. トラブルシューティング

### Q1. ImportExportTools NGがインストールできない

**A**: Thunderbirdのバージョンが古い可能性があります。
1. メニュー **≡** → **ヘルプ** → **Thunderbirdについて**
2. 最新版でない場合は更新してください

### Q2. エクスポートが途中で止まる

**A**: メールデータが大きすぎる可能性があります。
1. フォルダを小分けにしてエクスポート
2. 古いメールから順に移行

### Q3. Outlookへのドラッグ＆ドロップができない

**A**: 以下を試してください。
1. Outlookを**管理者として実行**
2. 一度に移行するファイル数を減らす（50件程度）
3. PCを再起動して再試行

### Q4. 移行後のメールが文字化けする

**A**: 古いメールで文字エンコードが異なる場合があります。
1. メールを開く
2. メッセージの文字化け部分を確認
3. 重要なメールは元のThunderbirdで参照

### Q5. 移行に時間がかかりすぎる

**A**: 以下を試してください。
1. 本当に必要なメールのみ選別して移行
2. 直近1年分のみ移行し、古いメールはThunderbirdで参照
3. 夜間や休日に実行

## 参考情報

### Thunderbirdのメール保存場所

Thunderbirdのメールデータは以下に保存されています:

```
C:\Users\<ユーザー名>\AppData\Roaming\Thunderbird\Profiles\<プロファイル名>\
```

### 移行を行わない選択

過去メールの移行は**任意**です。以下の理由で移行しないことも選択肢です:

- 新しいメールはOutlookで送受信できる
- 過去メールはThunderbirdで参照できる
- 移行作業に時間がかかる
- 古いメールを整理する良い機会

> **💡 推奨**: 重要なメールのみ移行し、その他はThunderbirdで参照する運用がおすすめです。
